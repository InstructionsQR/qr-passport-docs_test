# Часто задаваемые вопросы
## Процесс настройки и регистрация

{% cut "_Почему у меня неактивна кнопка «Зарегистрировать изделие»?_" %}

После создания модели возможность регистрации изделий временно недоступна. 
Это связано с тем, что администратору системы QR-Passport необходимо связать динамические параметры с шаблоном технического паспорта. Обычно этот процесс занимает не более часа. 

{% note info %}

Как только привязка будет выполнена, кнопка **Зарегистрировать изделие** станет активной.

{% endnote %}

{% endcut %}

{% cut "_Как динамические параметры (например, серийный номер) появляются в шаблоне технического паспорта?_" %}

После создания вами модели администратор системы с помощью специальных идентификаторов связывает динамические поля модели с пустыми полями в вашем шаблоне паспорта (PDF-файле). Именно на этом этапе данные (например, серийный номер) «подставляются» в итоговый паспорт для каждого конкретного изделия.

{% note warning %}

Рекомендуется загружать шаблоны паспортов в формате PDF – другие форматы могут нарушить корректную привязку динамических параметров к документу.

{% endnote %}

{% endcut %}

{% cut "_Что будет, если я в готовой модели заменю паспорт?_" %}

При замене паспорта кнопка **Зарегистрировать изделие** снова станет неактивной. 
Регистрация новых изделий по обновленному паспорту будет невозможна до тех пор, пока администратор системы QR-Passport не выполнит привязку полей заново.

{% note warning %}

Все изделия, зарегистрированные с предыдущим паспортом, останутся доступными по своим QR-кодам и будут ссылаться на старую версию документа. 
Новые изделия, зарегистрированные после замены, будут использовать обновленный паспорт. 

{% endnote %}

{% endcut %}

## Количественные параметры 

{% cut "_Сколько моделей я могу создать в системе QR-Passport?_" %}

Количество доступных для создания моделей зависит от вашего тарифного плана:

* Тариф **Старт**: до 3 моделей.
* Тарифы **Базовый** и **Премиум**: без ограничений.

{% endcut %}

{% cut "_Сколько изделий я могу зарегистрировать в одной модели?_" %}

Регистрация изделий для одной модели не ограничена. 

{% endcut %}

{% cut "_Сколько групп я могу создать в системе?_" %}

Создание групп в системе не ограничено.

{% endcut %}

## Удаление данных

{% cut "_Что будет, если я удалю изделие?_" %}

После удаления изделия из системы его QR-код становится неактивным, а все данные об изделии удаляются. 

{% endcut %}

{% cut "_Могу ли я удалить модель? Что произойдет с её изделиями?_" %}

Модель можно удалить, только если ни по одному из её изделий не было зафиксировано сканирований QR-кода. 
Даже одно сканирование переводит изделие в статус действующего и блокирует возможность удаления модели.
Если сканирований не было, то при удалении модели все привязанные к ней изделия и их QR-коды будут удалены.

{% endcut %}
